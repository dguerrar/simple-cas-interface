CASParameters

Parameters object to specify configuration when creating a new CAS object.
Properties:
Name Type Attributes Default Description
serverUrl string The base URL for the CAS server to authtenticate against, e.g https://example.com/cas.
serviceUrl string The URL to pass to the CAS server to indicate where clients should be redirected to after authentication, e.g https://my.special.app/casAuthResponseHandler.
protocolVersion number <optional>
 2.0 The CAS protocol version to use for communicating with the remote CAS server. Default: 2.0
method string <optional>
 GET The method the remote CAS server should use when redirecting clients back to the provided serviceUrl. Default: GET
